Kali Linux Training Module 01 PASSWORD CRACKING

 

Why exploit local passwords?

         To escalate privileges on the local host

         Local passwords can be used to gain access to other systems on the network

What is Cracking a Password?

         De-obfuscating a passwords representation

Brute force attack:

         Using all possible character combinations till a match for the password is found. Also known as an incremental attack in John the Ripper

Hashing

         A hash function takes an input (or password) and returns a fixed-size alphanumeric string

         Many different hashing algorithms exist such as MD2, SHA-2, LM, Cisco PIX

         You cannot reverse the hash to re-create the original file. Passwords are often stored as hashes instead of storing the actual password. Additionally, applications often salt passwords with extra characters before hashing them.
